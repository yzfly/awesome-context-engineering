## English

Thanks for your interest in improving **Awesome Context Engineering**! Contributions of papers, tools, articles, and translations are all welcome.

### Inclusion Criteria

A resource should meet the following before being added:

1. **Relevant** — directly related to context engineering for LLMs or AI agents (retrieval, processing, management, compression, isolation, memory, etc.).
2. **High quality** — well-known, actively maintained, or otherwise valuable to practitioners and researchers.
3. **Accessible** — the link is public and returns a valid page (no dead links, no paywalled-only content where avoidable).
4. **Not duplicated** — not already listed in the README.

### How to Contribute

1. Fork the repository and create a branch.
2. Add your resource to the appropriate section in both `README.md` (English) and `README_CN.md` (Chinese) to keep the two versions in sync.
3. Follow the existing formatting and ordering of each section.
4. Make sure all links work.
5. Open a Pull Request with a short description of what you added and why it belongs.

### Contribution Categories

- 📄 Research Papers
- 🛠️ Tools & Libraries
- 📖 Articles & Tutorials
- 💡 Expert Insights
- 🔧 Implementation Examples
- 🌐 Translations

## 中文

感谢你愿意为 **Awesome Context Engineering** 做出贡献！欢迎提交论文、工具、文章和翻译。

### 收录标准

资源在被收录前应满足以下条件：

1. **相关性** —— 与 LLM 或 AI 智能体的 context 工程直接相关（检索、处理、管理、压缩、隔离、记忆等）。
2. **高质量** —— 知名、活跃维护，或对从业者和研究者具有明确价值。
3. **可访问** —— 链接公开且能正常打开（无死链，尽量避免完全付费墙内容）。
4. **不重复** —— README 中尚未收录。

### 贡献流程

1. Fork 本仓库并创建分支。
2. 将资源添加到 `README.md`（英文）和 `README_CN.md`（中文）的相应章节，保持两个版本同步。
3. 遵循各章节已有的格式与排序。
4. 确保所有链接可用。
5. 提交 Pull Request，并简要说明你添加了什么以及为何应被收录。

### 贡献类别

- 📄 研究论文
- 🛠️ 工具与库
- 📖 文章与教程
- 💡 专家见解
- 🔧 实现示例
- 🌐 翻译
